# 📈 Stock Market Trend Analysis & Visualization

## 📌 Project Overview
Analyzed 4 years of historical stock data (2020–2024) for 5 major companies using Python to identify trends, patterns and investment insights.

## 🏢 Stocks Analyzed
- 🍎 Apple (AAPL)
- 🔍 Google (GOOGL)
- ⚡ Tesla (TSLA)
- 💻 Microsoft (MSFT)
- 📦 Amazon (AMZN)

## 🛠️ Tools Used
- Python
- Pandas & NumPy
- Matplotlib & Seaborn
- Plotly
- yfinance
- Google Colab
- GitHub

## 📊 Analysis Performed
- Exploratory Data Analysis (EDA)
- Moving Averages (20, 50, 200 day)
- Bollinger Bands
- RSI Indicator
- Volatility & Risk Analysis
- Correlation Analysis
- Cumulative Returns
- Candlestick Charts
- Summary Dashboard

## 🔍 Key Insights
- TSLA had highest return over 4 years
- MSFT was the safest stock (lowest volatility)
- Tech stocks are highly correlated
- COVID crash visible in March 2020
- All stocks recovered strongly after 2020

- stock_visuals.zip — All charts and graphs
